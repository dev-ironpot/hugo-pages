### Configuration File

```
├── config
│   ├── _default
│   │   ├── config.toml
│   │   ├── languages.toml
│   │   ├── menus.en.toml
│   │   ├── menus.zh.toml
│   │   └── params.toml
│   ├── production
│   │   ├── config.toml
│   │   └── params.toml
│   └── staging
│       ├── config.toml
│       └── params.toml

```

위 디렉토리 구조의 경우,

```bash
hugo --environment staging
```

위 명령어를 실행할 경우,config/\_default의 모든 설정을 사용하고 staging의 설정을 병합하게 됩니다.
[Configure Hugo](https://gohugo.io/getting-started/configuration/)
